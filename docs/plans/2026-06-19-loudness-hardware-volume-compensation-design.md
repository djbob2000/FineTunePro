# Дизайн автоматической компенсации громкости при переключении Loudness

Этот документ описывает дизайн автоматической компенсации воспринимаемой громкости звука на системном уровне macOS при включении и выключении функции Loudness Compensator.

## Проблема
При включении компенсации Loudness цифровой тракт ослабляет сигнал на величину пикового усиления фильтра (`headroomToSubtract`), чтобы избежать цифрового клиппинга. Это приводит к заметному падению воспринимаемой громкости средних частот в момент включения.

## Решение
При включении компенсации Loudness аппаратная/системная громкость активного устройства повышается на величину сдвига, компенсирующего цифровое ослабление. При выключении компенсации аппаратная громкость уменьшается на эту же величину.

Компенсация применяется только к устройствам с аппаратным (`.hardware`) или DDC (`.ddc`) контролем громкости.

## Алгоритм работы

### 1. Состояние в `AudioEngine.swift`
Мы добавляем словарь для отслеживания примененных сдвигов к каждому аудиоустройству по его UID:
```swift
private var appliedLoudnessOffsets: [String: Float] = [:]
```

### 2. Расчет сдвига (`offsetScalar`)
Сдвиг рассчитывается по формуле:
`offsetScalar = peakDB / 100.0`
Где `peakDB` — это пиковое усиление фильтров компенсации Loudness на текущей громкости системы и частоте дискретизации устройства (исключая частоты ниже 30 Гц). Поскольку в шкале громкости macOS 1 дБ соответствует изменению скаляра громкости на 0.01, деление на 100 дает точный сдвиг в шкале от 0.0 до 1.0.

### 3. Включение Loudness
1. Читаем текущую громкость устройства `V_curr`.
2. Вычисляем `peakDB` на основе `V_curr` и текущей частоты дискретизации.
3. Рассчитываем сдвиг `offsetScalar = peakDB / 100.0`.
4. Сохраняем `appliedLoudnessOffsets[deviceUID] = offsetScalar`.
5. Устанавливаем системную громкость в `min(1.0, V_curr + offsetScalar)`.

### 4. Выключение Loudness
1. Читаем текущую громкость устройства `V_curr`.
2. Извлекаем сохраненный сдвиг `offsetScalar = appliedLoudnessOffsets[deviceUID] ?? 0.0`.
3. Сбрасываем сохраненный сдвиг `appliedLoudnessOffsets[deviceUID] = 0.0`.
4. Устанавливаем системную громкость в `max(0.0, V_curr - offsetScalar)`.
